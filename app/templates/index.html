Insider Threat Detection System

Welcome! Follow the instructions below to train models and detect anomalies using Postman.

Instructions for Training and Detecting Models

Step 1: Prepare Your Data

For each model, prepare your training data in JSON format as shown in the examples below.

Step 2: Use Postman to Make Requests

Select the appropriate endpoint for training or detection from the list below.

Use POST as the request type.

Step 3: Enter the URL in Postman

For training, use the following base URL:

http://localhost:5000/train_end_point_

For detecting anomalies, use:

http://localhost:5000/detect_end_point_anomalies

Ensure the correct endpoint matches your selected operation.

Step 4: Input JSON Data

In the Body tab, select raw and choose JSON from the dropdown. Then, input your JSON data.

Step 5: Send Your Request

Click Send to execute the request. Check the response for success or error messages.
